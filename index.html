01 MÚSICA

LISTA DE COMANDOS

A MÚSICA
URL
/play
Toca por URL do YouTube ou busca automática via YouTube Data API v3.
CTRL
/skip
Pula para a próxima música. Fila vazia — bot para e sai do canal.
HIST
/previous
Volta para a anterior. Funciona após /stop graças ao estado persistente.
CTRL
/pause
Pausa sem sair do canal. Estado da fila preservado.
CTRL
/resume
Retoma de onde parou. Só funciona se estiver em pausa.
CTRL
/stop
Para, limpa fila, sai do canal. Última música fica salva no estado.
B MODERAÇÃO
MOD
/kick
Remove o usuário do servidor. Requer permissão de Kick Members.
MOD
/ban
Bane permanentemente do servidor. Suporte a motivo e delete-messages.
VOICE
/mute
Cala o usuário no canal de voz. Suporte a timeout no servidor.
VOICE
/unmute
Libera o silêncio do usuário no canal de voz.
WARN
/warn
Adiciona um aviso ao registro do usuário com motivo.
WARN
/viewwarn
Lista os avisos de um usuário com data, motivo e moderador.
CTRL
/clear
Limpa de 1 a 100 mensagens de um canal. Filtra por usuário se quiser.
CTRL
/unban
Remove o ban de um usuário. Funciona com username ou ID.

04 SOBRE

SOBRE O PROJETO

Trinity é um bot de música para Discord construído do zero em Node.js, desenvolvido com foco em aprendizado — cada linha de código entendida antes de ser escrita.

Evoluiu de ytdl-core para play-dl e finalmente para DisTube + yt-dlp, a solução estável para autenticação com o YouTube.

Suporte a YouTube (URL e busca via YouTube Data API v3). Arquitetura orientada a eventos com discord.js v14 e carregamento automático de comandos por subpasta — tanto os de música quanto os de moderação (kick, ban, mute, warn, clear, unban, viewwarn).

Node.js discord.js v14 DisTube v5 yt-dlp ffmpeg-static opusscript axios dotenv
